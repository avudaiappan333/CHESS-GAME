# Chess Game

This is a chess game created using HTML, CSS, and JavaScript.

1.Features
- Chess board
- Piece movement
- Turn based game
- Basic check detection

2. Technologies Used
- HTML
- CSS
- JavaScript

3. How to Run
. Download the project.
. Open the folder.
. Open index.html in a browser.
